## 📌 Overview

A sleek and modern digital clock built using HTML, CSS, and JavaScript. The project features real-time time updates, AM/PM display, colorful UI elements, and a beautiful reflection effect created entirely with CSS.

---

## ✨ Features

✅ Real-Time Digital Clock

✅ 12-Hour Time Format

✅ AM / PM Indicator

✅ CSS Reflection Effect

✅ Modern Neon Design

✅ Responsive Layout

✅ Pure HTML, CSS & JavaScript

---
